12/10/2004 17:59:03 added API for attachments there is a method (getAttachmentManager) on the model class which will return a CAttachmentManager for a given object.

the CAttachmentManager class has a whole set of methods for manipulating the attachments for an object:


count
the number of attachments

forObject
return the object that the attachment manager is for

addAttachment
add a path to the attachments which are being managed

addAttachment(thePath As String) As Boolean

amendAttachment
convert an attachment from one path to another

amendAttachment(oldPath As String, newPath As String) As Boolean

deleteAttachment
remove an attachment from the object
deleteAttachment(aPath As String) As Boolean
item
return the attachment path for item X

item(anItemID as long) as string
anItemID can be 1 to count

Example
-------------
set anAttachmentManager= cdiface.theModel.getAttachmentManager(cdiface.theobject)

msgbox "Now Adding some attachments"
anattachmentmanager.addattachment("C:\temp.txt")

msgbox "Now Amending some attachments"
call anattachmentmanager.amendattachment("C:\temp.txt","C:\temp2.txt")

msgbox "Now Delete some attachments"
call anattachmentmanager.deleteattachment("C:\temp2.txt")
