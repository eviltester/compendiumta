22/08/2004 16:54:46 added new API commands The API has a new item

theGUI

e.g. cdiface.theGUI

which has the following functions:

EditObject
EditObjectProperties
EditAsHierarchy
toggleAppBar

all of the will take any valid Compendium-TA object but will only open a window if it is valid for the function.



editObject will mainly open a properties window but for a graph or a path will open the editGraph and editPath dialogs.

EditObjectProperties will open a properties window to edit the object

EditAsHierarchy will display the object in a new hierarchy window if it can be displayed in such i.e. a hierarchy branch, a hierarchy, a graph

usage would be:

cdiface.theGUI.editObject anObject
cdiface.theGUI.EditObjectProperties anObject
cdiface.theGUI.EditAsHierarchy anObject



toggleAppBar will toggle the application control panel on or off. With no arguments it is a toggle, with a value of true it will show the application control panel and with a value of false it will hide the application control panel

e.g.

cdiface.theGUI.toggleAppBar
cdiface.theGUI.toggleAppBar true
cdiface.theGUI.toggleAppBar false
